## 📖 About

**ULP-WEB-SERCHER** reads combo files in `URL:EMAIL:PASS` format, searches for a specific app or website URL, and saves all matching results into a dedicated output file.

---

## ✨ Features

- 📂 **Load combo files** in `URL:EMAIL:PASS` format
- 🔎 **Search by app name or URL** (e.g. `facebook`, `https://facebook.com`)
- 💾 **Auto-save results** to `results_<app_name>.txt`
- ⚡ **Fast filtering** through large combo lists
- 📱 **Android-friendly** — runs on mobile

---

## 🚀 How to Use

1. **Load** your combo file (format: `URL:EMAIL:PASS`)
2. **Enter** your target app name or URL (e.g. `facebook` or `https://facebook.com`)
3. **Hit Search** — matching lines are filtered out
4. **Results** are saved automatically to `results_facebook.txt`

### 📋 Input Format Example
```
https://facebook.com/login:user@email.com:password123
https://instagram.com:anotheruser@gmail.com:pass456
https://facebook.com:test@mail.com:mypassword
```

### 📤 Output Example (`results_facebook.txt`)
```
https://facebook.com/login:user@email.com:password123
https://facebook.com:test@mail.com:mypassword
```
